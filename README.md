## 用户定位
系统的使用包含管理员、学生、校内商户
管理员主要由后勤处担任，负责商户的录入、文章的发布、员工的增加
商户负责人也具有后台管理的权限，不过仅能增加自身商户的员工
学生仅能访问门户系统，查看文章并投诉商户
用户信息大致包含：id、用户名、真实姓名、密码、所属部门、学号（管理员需要分级，学生需要增加学号字段）、用户级别（学生、负责人、员工、管理员）、手机号、邮箱等
